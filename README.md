Baku
====

No nonsense backup client.


Usage
=====

Command Line
------------

To use the command line of the program use baku-cli.exe. The input parameters are

* Source directory
* Destination directory
* File extensions


Example Usage
-------------

`baku-cli.exe "C:\source" "D:\destination" jpg png doc* ppt* xls*`


Graphical User Interface
------------------------

There is nothing to explain here, the usage is quite straight forward.
